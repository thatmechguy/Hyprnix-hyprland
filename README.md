#### Default Keybinds:
- **SUPER + D** = Open Menu / Launcher
- **SUPER + E** = Open File Manager (Nautilus)
- **SUPER + RETURN** = Open Terminal (Kitty)
- **SUPER + Q** = Close Active Window
- **SUPER + B** = Open Web Browser (Helium)
- **SUPER + SHIFT + T** = Open System Monitor (btop)
- **SUPER + T** = Toggle Floating Mode
- **SUPER + F** = Toggle Maximized State
- **SUPER + SHIFT + F** = Toggle Fullscreen State
- **SUPER + C** = Center Window
- **SUPER + TAB** = Focus Previous Workspace
- **ALT + TAB** = Cycle Next Window
- **SUPER + PERIOD** = Open Emoji Picker
- **SUPER + Y** = Toggle Wallpaper Panel

#### Navigation & Workspace Management:
- **SUPER + Arrow Keys** = Change window focus (Left/Right/Up/Down)
- **SUPER + SHIFT + Arrow Keys** = Swap tiled window position (Left/Right/Up/Down)
- **SUPER + [1 - 0]** = Switch to Workspace 1-10 (0 key maps to workspace 10)
- **SUPER + SHIFT + [1 - 0]** = Move active window to Workspace 1-10
- **SUPER + Mouse Scroll Up/Down** = Switch to next/previous workspace

#### Mouse Shortcuts:
- **SUPER + Left Click (Hold & Drag)** = Move/Drag window
- **SUPER + Right Click (Hold & Drag)** = Resize window

#### Screenshots & Media:
- **PRINT** = Take screenshot of a selected region
- **SUPER + PRINT** = Take fullscreen screenshot
- **XF86AudioPlay** = Media Play/Pause
- **XF86AudioNext/Prev** = Next/Previous track
- **XF86AudioRaise/LowerVolume** = Increase/Decrease Volume
- **XF86AudioMute** = Mute audio output
- **XF86MonBrightnessUp/Down** = Increase/Decrease brightness

#### System & Debugging:
- **SUPER + F12** = Toggle debug overlay
- **Lid Switch (Close)** = Lock screen and set power profile to balanced

#### Clone the repository:
```
git clone --depth 1 https://github.com/thatmechguy/Hypnix-hyprland.git
```

#### To install the configuration using the install script, you must clone the repo in the home directory
```
cd ~/Hyprnix-hyprland
chmod +x ./install.sh
./install.sh
```
